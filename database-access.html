Research Database Access

The Applied Science Research Institute maintains a comprehensive database of application analyses, behavioral studies, and research findings. Our database is available for academic and research purposes, supporting the scientific community in advancing understanding of application behaviors.

2.3M Applications Analyzed
500K+ Network Analyses
87% Over-Permissioning Rate

Database Contents

Our research database includes:

  • Application Metadata: Comprehensive information about applications including categories, permissions, and basic characteristics
  • Permission Analysis: Detailed analysis of permission requests and usage patterns
  • Network Communication Data: Network traffic patterns, protocol usage, and communication endpoints
  • Behavioral Studies: User behavior tracking mechanisms and data collection practices
  • Research Findings: Aggregated results from our published and ongoing research
  • Anonymized Survey Data: User awareness studies and privacy perception data

Access Requirements

Academic and Research Use: Our database is primarily intended for academic research, educational purposes, and non-commercial scientific inquiry.

Eligibility

Database access is available to:

  • Researchers affiliated with academic institutions
  • Graduate students conducting research
  • Non-profit research organizations
  • Government research agencies
  • Other qualified researchers (case-by-case review)

Application Process

To request database access, please provide:

  1. Research Purpose: Clear description of your research objectives and how the data will be used
  2. Institutional Affiliation: Your academic or research institution
  3. Data Requirements: Specific datasets or data types needed for your research
  4. Ethical Approval: Confirmation of institutional review board approval (if applicable)
  5. Contact Information: Email address and institutional contact details

Data Access Request

Please send your data access request to research@appresearch.org with the subject line "Database Access Request" and include all required information listed above.

Data Usage Terms

All database users must comply with our Data Usage Policy. Key requirements include:

  • Use data only for stated research purposes
  • Provide proper attribution in publications
  • Maintain data security and confidentiality
  • Do not attempt to re-identify anonymized data
  • Report any data security incidents
  • Comply with applicable privacy laws and regulations

Available Datasets

Permission Analysis Dataset

Comprehensive analysis of permission requests across 2.3 million applications, including permission patterns, category correlations, and over-permissioning analysis.

Network Communication Dataset

Network traffic analysis data from 500,000+ applications, including protocol usage, encryption practices, and third-party communication patterns.

User Awareness Survey Data

Anonymized survey responses from 5,000+ participants regarding data collection awareness, privacy perceptions, and application usage patterns.

Application Behavior Dataset

Behavioral analysis data including tracking mechanisms, data collection practices, and user interaction patterns.

Data Formats

Data is available in multiple formats:

  • CSV: Structured data for statistical analysis
  • JSON: Hierarchical data structures
  • SQL Dumps: Database exports for direct import
  • API Access: Programmatic access for large-scale research (limited availability)

Processing Time

Data access requests are typically processed within 5-10 business days. Complex requests or requests requiring additional review may take longer. You will receive email confirmation when your request is approved and data access is granted.

Support
